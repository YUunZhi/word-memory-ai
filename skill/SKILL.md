AI 单词记忆个性化规划器

一、功能介绍

本技能基于大模型自然语言理解能力，结合艾宾浩斯遗忘曲线，根据用户录入单词、单词掌握程度，动态生成每日个性化背诵清单，自动生成配套自测练习题。传统表格、普通背单词软件只能固定词库，无法自适应每个人的遗忘情况，该智能规划逻辑必须依靠 AI 实现。

二、输入输出说明

输入内容

单词列表：中英文单词

掌握等级：完全陌生 / 有点印象 / 熟练掌握

每日可背诵时长

输出内容

今日分层背诵计划（生词优先复习，熟词少量巩固）

单词自测填空题、选择题

薄弱词汇标注

三、运行依赖

Python 3.10

大模型 API 密钥（DeepSeek / 豆包 API）

基础网络环境

四、技能配置 yaml

yaml

skill\_name: AI单词记忆个性化规划器

version: 1.0

author: 你的名字

input\_params:

&#x20; word\_list: 单词文本

&#x20; master\_level: 掌握程度分级

&#x20; study\_time: 每日学习时长

output\_content:

&#x20; daily\_plan: 每日背诵规划

&#x20; test\_paper: 单词自测习题

&#x20; weak\_words: 个人薄弱单词汇总

五、使用方法

运行 scripts 文件夹下 main.py 主程序

按提示输入单词与掌握情况

AI 自动生成专属背词方案并打印输出
